
# 🧠 Sistema de Avaliação Neuropsicológica

Sistema desenvolvido para auxiliar no gerenciamento de avaliações neuropsicológicas, permitindo o cadastro de pacientes, registro de avaliações e consulta de resultados de forma prática e organizada.

## Funcionalidades

- Cadastro de pacientes
- Registro de avaliações
- Consulta de histórico
- Visualização de resultados
- Interface intuitiva e responsiva

## Tecnologias

- HTML
- CSS
- JavaScript
